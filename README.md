### 功能
* 页面板块  
    * 基本网页内容分页显示、浏览、跳转
    * 分类模糊搜索资讯、教程，实时显示
    * 分类模糊搜索车队成员 


* 用户板块
    * 一般账号登录、注册
    * 记住我功能
    * 获取天气和问候
    * 主页上传头像
    * 申请成为管理员
    * 更新、重置资料、密码，表单检验
    * 消息中心获取个人消息


* 管理板块
    * 管理账号登录
    * 获取天气和问候
    * 进入时提示有未处理消息
    * 管理员账号分页显示、基本CRUD和表单检验
    * 账号禁用检测，无法禁用最后一个启用的管理员
    * 管理员账号模糊搜索，实时显示
    * 用户账号分页显示、基本CRUD和表单检验
    * 用户账号模糊搜索，实时显示
    * 消息中心显示管理员申请消息
    * 车队管理成员分页显示、基本CRUD和表单检验
    * 成员模糊搜索，实时显示
